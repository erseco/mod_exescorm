Description of test packages
============================================

Sample Packages downloaded from http://exescorm.com/exescorm-explained/technical-exescorm/golf-examples/

* singlescobasic.zip - Single SCO with basic runtime calls. SCORM 1.2.
* singlesco_exescorm12.zip - Single SCO content packaging example. SCORM 1.2.
* RuntimeMinimumCalls_EXESCORM12.zip - Multi-SCO packaging example. SCORM 1.2.
* RuntimeBasicCalls_EXESCORM20043rdEdition.zip - Multi-SCO packaging example. SCORM 2004 3rd edition.
* RuntimeMinimumCalls_EXESCORM12-mini.zip - modified 2 SCO version of RuntimeMinimumCalls_EXESCORM12.zip

These packages were downloaded from http://exescorm.com/ website,  the website
disclaimer states that *Content on this site is licensed under a Creative
Commons Attribution 3.0 License*. http://creativecommons.org/licenses/by/3.0/


Other test packages
* badexescorm.zip - contains a fake imsmanifest.xml inside a directory, used for validation check.
* invalid.zip - zip file with an single html file, no EXESCORM config files, used for validation check.
* validexescorm.zip - non functional package with an imsmanifest.xml, used for validation check.
* validaicc.zip - non functional package with AICC config files, used for validation check.
* complexexescorm.zip - copied from: https://github.com/jleyva/exescorm-debugger.
* singlescobasic_missingorg.zip - copy of exescorm.com package but with missing org definition.
